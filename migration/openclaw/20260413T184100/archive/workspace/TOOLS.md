# TOOLS.md - Local Notes

Skills define _how_ tools work. This file is for _your_ specifics — the stuff that's unique to your setup.

## Search & Research

### Tavily (Primary)
- **Use for**: Web search, real-time information, fact-checking
- **API Key**: Set via `TAVILY_API_KEY` environment variable
- **Skill**: `tavily` installed in workspace/skills/
- **When to use**: Current events, documentation lookups, verifying claims, researching unfamiliar topics

### Answer Overflow
- **Use for**: Discord community discussions, coding Q&A that only exists in Discord
- **Skill**: `answeroverflow` installed in workspace/skills/
- **When to use**: Library-specific issues, community solutions, debugging patterns from Discord communities
